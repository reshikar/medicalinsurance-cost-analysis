# Medical Insurance Cost Analysis using R

Author: Reshika Rimal  

## Project Overview
This project explores the relationship between age, geographic region, and medical insurance costs using a real dataset of insured individuals in the United States.

## Objectives
- Perform exploratory data analysis (EDA)
- Investigate relationships between predictors and medical costs
- Fit a linear regression model
- Evaluate model performance using Mean Squared Error (MSE)

## Variables Used
- Age
- Region
- Medical insurance charges

## Methods
- Scatter plots and LOWESS smoothing
- Summary statistics
- Train/Test split (80/20)
- Linear regression model
- Model performance evaluation

## Key Findings
- Medical insurance cost increases with age.
- Regional differences exist, with slightly higher costs in the Southeast.
- Age is a statistically significant predictor of insurance costs.

## Model Results
Training MSE: 139,760,430  
Test MSE: 103,676,931

## Tools
- R
- Linear Regression
- Statistical Learning
